# Privacy

This plugin does not collect, store, or transmit any personal data.

The skills run entirely within your Claude Code session. They read files from your repository, invoke subagents, and produce output locally. Nothing is sent to the plugin author.

Any network activity (web searches, API calls, MCP servers) is performed by Claude Code itself under your own account and configuration, subject to [Anthropic's privacy policy](https://www.anthropic.com/legal/privacy) and the policies of any third-party tools you have configured.
